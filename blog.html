Question-1: What are the different ways to select an element in the DOM?

1. getElementById() – আইডি দিয়ে এলিমেন্ট সিলেক্ট করা। 2. getElementsByClassName() – ক্লাস নাম ব্যবহার করে একাধিক এলিমেন্ট সিলেক্ট করা। 3. getElementsByTagName() – নির্দিষ্ট ট্যাগ নাম দিয়ে এলিমেন্ট সিলেক্ট করা। 4. querySelector() – CSS সিলেক্টর ব্যবহার করে প্রথম ম্যাচিং এলিমেন্ট সিলেক্ট করা। 5. querySelectorAll() – CSS সিলেক্টর দিয়ে সব ম্যাচিং এলিমেন্ট সিলেক্ট করা।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML – এলিমেন্টের মধ্যে থাকা HTML কোডসহ সমস্ত কনটেন্ট রিটার্ন করে। 2. innerText – শুধুমাত্র ভিজিবল টেক্সট রিটার্ন করে। 3. textContent – পুরো টেক্সট রিটার্ন করে, HTML ট্যাগ ছাড়া।

Question-3: What is event delegation in the DOM?

Event Delegation যেখানে কোনো parent এলিমেন্টে ইভেন্ট listener যোগ করা হয়, যা তার child এলিমেন্টগুলোর ইভেন্ট হ্যান্ডল করতে পারে।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো একটি প্রসেস, child এলিমেন্টে ইভেন্ট ঘটলে, সেটি parent এবং ancestor এলিমেন্টগুলোর কাছেও পৌঁছে যায়। এটি ডিফল্টভাবে কাজ করে এবং stopPropagation() ব্যবহার করে থামে

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create Element: document.createElement('tagName') দিয়ে নতুন এলিমেন্ট তৈরি করা হয়। 2. Add Element: appendChild(), append(), বা insertBefore() ব্যবহার করে এলিমেন্ট যোগ করা যায়। 3. Remove Element: removeChild() বা remove() ব্যবহার করে এলিমেন্ট মুছে ফেলা যায়।
